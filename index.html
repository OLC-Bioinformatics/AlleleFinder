STEC AlleleFinder

STEC AlleleFinder is a suite of Python tools designed for the discovery, sequence typing, and profiling of the stx alleles of Shiga toxin-producing E. coli.

Installation

Detailed installation instructions for STEC AlleleFinder are provided in the Installation Guide.

Scripts

STEC AlleleFinder comprises a single script with six separate functionalities:

  1. Profile Reduce
  2. Allele Translate Reduce
  3. Allele Find
  4. Allele Translate Find
  5. AA Allele Find
  6. Allele Split
  7. Allele Concatenate
  8. STEC Combined Subunits

Each link will take you to a detailed description of each functionality.

Feedback

If you encounter any issues while installing or running AlleleFinder, have feature requests, or need assistance, please open an issue on GitHub.
