STEC AlleleFinder

A suite of tools, written in Python, designed for the discovery, sequence typing, and profiling the stx alleles of Shiga toxin-producing E. coli

Installation

Detailed installation instructions are provided here

Scripts

There is a single STEC script with five separate functionalities:

  1. profile_reduce
  2. allele_translate_reduce
  3. allele_find
  4. aa_allele_find
  5. allele_split
